## ✅ 5줄 그래프 암기 세트

```python
import matplotlib.pyplot as plt   # ① 그래프 도구 불러오기
x = range(100)                    # ② x 값 만들기
y = x                             # ③ y 값 만들기 (예: y=x)
plt.plot(x, y)                    # ④ 그리기
plt.show()                        # ⑤ 화면에 보여주기
```

---

## 줄별 초압축 암기 문장

1️⃣ **import** → 그래프 준비
2️⃣ **x** → 가로축
3️⃣ **y** → 세로축
4️⃣ **plot** → 그림
5️⃣ **show** → 출력
