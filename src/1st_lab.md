아주 좋은 주제입니다.
**“파이썬 + IDLE + 초보자 + 실습 10개”** 조합은 입문 수업의 정석입니다.

아래는 **완전 초보자가 IDLE에서 바로 따라 치며 실습할 수 있는
10가지 코딩 실습 세트**입니다.
(※ 모두 **외우기 가능한 수준**, GPT와 함께 쓰기 좋게 구성)

---

## 🧩 파이썬 IDLE 초보자용 10가지 실습 코딩

---

### ① 두 수를 더하여 출력하기 ✅

```python
a = int(input("첫 번째 수: "))
b = int(input("두 번째 수: "))
print("합:", a + b)
```

**배우는 것**: 입력, 정수 변환, 출력

---

### ② 두 수의 사칙연산 출력

```python
a = int(input())
b = int(input())

print(a+b, a-b, a*b, a//b)
```

**배우는 것**: 연산자

---

### ③ 문자열 출력하기 (Hello Python)

```python
print("Hello Python")
```

**배우는 것**: 가장 첫 성공 경험

---

### ④ 반복 출력 (1부터 5까지)

```python
for i in range(1, 6):
    print(i)
```

**배우는 것**: 반복문

---

### ⑤ 별(*) 찍기

```python
for i in range(1, 6):
    print("*" * i)
```

**배우는 것**: 반복 + 문자열

---

### ⑥ 리스트 만들고 출력

```python
nums = [1, 2, 3, 4, 5]
print(nums)
```

**배우는 것**: 리스트 개념

---

### ⑦ 리스트 합 구하기

```python
nums = [1, 2, 3, 4, 5]
print(sum(nums))
```

**배우는 것**: 내장 함수

---

### ⑧ 조건문 (짝수/홀수 판별)

```python
n = int(input())
if n % 2 == 0:
    print("짝수")
else:
    print("홀수")
```

**배우는 것**: if / else

---

### ⑨ 함수 만들기

```python
def add(a, b):
    return a + b

print(add(3, 4))
```

**배우는 것**: 함수 개념

---

### ⑩ 선형 곡선 그리기 📈 (가장 인기)

```python
import matplotlib.pyplot as plt

x = range(10)
y = x

plt.plot(x, y)
plt.show()
```

**배우는 것**: 시각화, 수학적 직관

---

## 🎯 초보자 수업 운영 팁 (중요)

### ✔ 실습 원칙 3가지

1. **코드는 반드시 직접 타이핑**
2. **에러 메시지도 GPT에게 질문**
3. **한 줄씩 바꿔보게 하기**

### ✔ GPT 질문 예시

> “이 코드에서
> 한 줄만 바꾸면
> 결과가 어떻게 달라져?”
