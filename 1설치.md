---

## ① 파이썬 다운로드 (가장 중요)

1️⃣ 인터넷에서 **python.org** 검색
2️⃣ 화면에 보이는 **Download Python** 버튼 클릭
3️⃣ 파일 다운로드 완료 → **더블 클릭**

> 💡 *아무 설정 안 바꿔도 됩니다. 단, 아래 체크만 필수!*

---

## ② 설치할 때 이것만 꼭 하세요 ⭐

설치 화면에서 **맨 아래 체크박스**를 찾으세요.

☑ **Add Python to PATH**  ← 반드시 체크
👉 그리고 **Install Now** 클릭

> ✔ 이거 하나만 체크하면
> ✔ 나중에 문제 생길 일이 거의 없습니다

---

## ③ 파이썬 실행 (IDLE 열기)

1️⃣ 키보드 **윈도우 키** 누르기
2️⃣ **IDLE** 검색
3️⃣ **IDLE (Python 3.x)** 클릭

이 화면이 나오면 성공입니다 👇

```
>>>
```

---

## ④ 첫 실행 (이 줄만 쳐보세요)

```python
print("Hello Python")
```

Enter 치면 👇

```
Hello Python
```

🎉 **설치 + 실행 끝!**

---

## 🧠 완전 초보자 암기용 1문장

> **다운로드 → PATH 체크 → IDLE → print 한 줄**

---

## ❗ 초보자가 가장 많이 하는 실수 2가지

❌ PATH 체크 안 함
❌ Python이 아니라 다른 프로그램 실행

👉 **IDLE만 열면 90% 해결**
